Floating Image App

אפליקציה שקופה להצגת תמונות, PDF, טקסט ועוד, מעל כל תוכנה – בעיצוב יוקרתי.
מתאימה למעצבים, מפתחים ואמנים שרוצים השראה "צפה" על המסך מבלי להפריע לעבודה.

✨ פיצ'רים

- ✅ תמיכה בתמונות (JPG, PNG, GIF...)
- ✅ תמיכה ב־PDF
- ✅ תמיכה בקבצי טקסט (TXT)
- ✅ תמיכה בסיסית בקבצי מצגת (PPT, PPTX – עם הודעת מידע)
- ✅ שינוי שקיפות התמונה
- ✅ שינוי גודל (scale)
- ✅ גרירה חופשית במסך
- ✅ נעילה/שחרור – כולל קיצורי מקשים וכפתור צף
- ✅ "מצב השראה" – כאשר נעול, התמונה נצמדת לסמן העכבר

🚀 הפעלה בפיתוח

```bash
npm install
npm start
```

🛠 בניית אפליקציה להרצה

```bash
npx electron-packager . FloatingImageApp \
  --platform=win32 \
  --arch=x64 \
  --icon=icon.ico \
  --overwrite \
  --out=dist
```

⌨ קיצורי מקשים

- `Ctrl + Shift + U` – שחרור תמונה
- `Ctrl + Shift + Q` – סגירת האפליקציה
- `Esc` – שחרור מהיר

🖼️ פורמטים נתמכים

- תמונות: `png`, `jpg`, `jpeg`, `gif`, `webp`
- מסמכים: `pdf`, `txt`
- מצגות: `ppt`, `pptx` (מציג הודעה בלבד)

📁 מבנה הקבצים

- `index.html` – ממשק משתמש והלוגיקה הראשית
- `main.js` – קובץ ראשי של Electron
- `package.json` – הגדרות הפרויקט
- `icon.ico` – אייקון מותאם אישית
